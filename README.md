# Privacy Policy

WeCampus does not require users to provide personally identifiable information to use the core features of the application.

The application does not require users to upload photos and does not access or use device features such as the microphone or camera.

WeCampus does not collect or process financial information, contact lists, health data, or other sensitive personal data. Additionally, the application does not request or track precise location data or residence information.

The app uses Supabase as its backend infrastructure solely for database and service functionality. Any data stored is limited to what is necessary for the core operation of the application.

WeCampus is not directed at children under the age of 13.

If you have any questions regarding this Privacy Policy, you can contact us at:
wecampus.support@gmail.com


# User-Provided Data

WeCampus allows users to voluntarily provide and share personal information, such as their name, surname, educational background, daily activities, and résumé details, in order to interact with other users on the platform.

Providing this information is entirely optional. Such data is only collected, processed, and stored when the user chooses to submit it.

The provided information may be visible to other users within the application as part of the platform’s social and discovery features.

By creating an account or using the application in a logged-in state, users acknowledge and consent to the processing, storage, and display of the information they choose to provide, in accordance with this Privacy Policy.

All user data is stored securely using Supabase infrastructure and is limited to what is necessary for the operation of the application.


# User Rights

Users have the right to access, update, or delete their personal data at any time within the application. If users wish to request deletion of their data, they may contact us at wecampus.support@gmail.com.


# Data Sharing

WeCampus does not sell, rent, or share user data with third parties for marketing or commercial purposes.

Data may only be processed by trusted service providers (such as Supabase) strictly for the purpose of operating the application.


# Data Security

We take reasonable technical and organizational measures to protect user data against unauthorized access, loss, or misuse.

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated effective date.
